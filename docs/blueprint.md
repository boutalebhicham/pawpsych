# **App Name**: PawPsych

## Core Features:

- Personality Quiz Interface: An intuitive, multi-step questionnaire for pet owners to describe their dog's or cat's behaviors and traits.
- AI-Powered Personality Profile Generation: Utilize an AI tool to analyze quiz responses and generate a detailed, personalized, and engaging personality profile for the pet.
- Shareable Results Display: Display the generated pet personality profile in a visually appealing format, with options for sharing on social media.
- Admin Question Management: A simplified administrative interface to add, edit, or remove questions and predefined answer options for the personality quiz.

## Style Guidelines:

- Primary color: A soft, cheerful yellow (#F5C033) evoking warmth and playfulness.
- Background color: A light, muted cream (#F9F5E6) providing a clean and inviting canvas.
- Accent color: A vibrant, warm orange (#F2460D) for calls to action and important highlights, providing an energetic contrast.
- Headlines: 'Poppins' (sans-serif) for a precise, contemporary, and engaging feel. Body text: 'PT Sans' (sans-serif) for clear readability and a touch of warmth.
- Use playful and friendly icons, featuring elements like paws, animal silhouettes, and cartoon-style bubbles.
- A clean, spacious, and responsive layout that is mobile-first, ensuring ease of use on all devices.
- Subtle and gentle animations for page transitions and when displaying quiz results, adding to a delightful user experience.
